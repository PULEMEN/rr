Редактировать пользователя

Укажите имя и адрес электронной почты для этого пользователя. Адрес электронной почты будет использоваться для входа в приложение.

Когда используется внешняя система аутентификации (например, SAML2, OIDC или LDAP), этот идентификатор будет использоваться для связывания пользователя BookStack с учетной записью системы аутентификации. Вы можете игнорировать это поле, если используете стандартную аутентификацию по электронной почте.

Назначьте роли пользователю. Если назначено несколько ролей, разрешения будут суммироваться и пользователь получит все права назначенных ролей.

Установите пароль для входа в приложение. Длина пароля должна быть не менее 8 символов.

Заполните поля ниже только если вы хотите изменить пароль.

Выберите изображение. Изображение должно быть квадратным, размером около 256px.


Этот параметр изменит язык интерфейса приложения. Это не влияет на созданный пользователем контент.
